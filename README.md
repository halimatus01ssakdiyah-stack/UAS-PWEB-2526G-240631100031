## Cara Menjalankan

1. Import file `database.sql` ke phpMyAdmin.
2. Simpan project di folder `htdocs`.
3. Jalankan Apache dan MySQL pada XAMPP.
4. Buka browser dan akses:

```
http://localhost/crud.uas/login.php
```
